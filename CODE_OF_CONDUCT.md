# Code of Conduct

## Purpose
Create a safe and professional collaboration environment for all contributors.

## Expected Behavior
- Be respectful, inclusive, and constructive.
- Focus discussions on technical and product outcomes.
- Provide actionable feedback with clear rationale.

## Unacceptable Behavior
- Harassment, discrimination, personal attacks, or hostile language.
- Sharing confidential data without authorization.
- Disruptive behavior that blocks healthy collaboration.

## Enforcement
Project maintainers may remove or reject contributions that violate this policy.
Severe or repeated violations may result in restricted participation.

## Reporting
Report incidents privately to maintainers or designated project contacts.
