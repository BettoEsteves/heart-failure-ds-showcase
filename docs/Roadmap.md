# Roadmap

## Phase 1 — Governance Baseline
- Finalize template structure and policy files.
- Enforce CI checks for structure and forbidden paths.

## Phase 2 — Multi-Language Enablement
- Add starter examples for Python, R, and Rust by project need.
- Introduce language-specific checks in CI as each stack is activated.

## Phase 3 — AI/LLM Operations
- Standardize local/cloud model integration guidance.
- Add evaluation and monitoring conventions for AI outputs.

## Phase 4 — Infrastructure Maturity
- Expand Terraform modules by environment.
- Add deployment controls, approvals, and release governance.
