# Skills Catalog

This template supports multi-language and AI-first project bootstrapping.

## Core Skills
- Project governance and repository hygiene
- Python data/ML baseline setup
- R/Posit analytical workflows
- Rust service and tooling foundations
- LLM integration patterns (local and cloud)
- CI/CD and infrastructure-as-code workflows

## Expected Operator Profile
- Product/engineering teams building repeatable, governed AI projects
- Teams requiring auditable setup and enforceable standards from day one

## Usage
Use these skills as capability tags when defining tasks and assigning ownership in `docs/BoardTasks.md`.
